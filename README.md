# SeriesHub

SeriesHub é um aplicativo mobile para usuários poderem utilizar como uma forma de registrar séries nas quais estão assistindo, querem assistir e/ou já assistiram.

## Tecnologias Utilizadas

- Flutter
- SQLite
- Firebase

### Plugins
+ SQFlite
+ Provider
+ ImagePicker
+ Firebase
  + Authenticator
